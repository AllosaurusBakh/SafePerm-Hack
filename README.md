## Задача
Реализовать сервис, который должен производить анализ и предсказание чрезвычайных происшествий на территории пермского края.

## Стек
### Frontend
- TypeScript
- React
- Vite

### Backend
- Python
- FastAPI

### Model
- Scikit-learn
- PyTorch

## Поднятие backend
### Требования
* Python 3.x, x >= 9
* venv

### Команды
```commandline
cd backhacks
pip install -r requirements.txt
uvicorn main:app --reload 
```

## Поднятие frontend
### Требования
* npm (менеджер пакетов)
* node.js
  
### Команды
```commandline
cd fronthacks
npm i
npm run dev
```
